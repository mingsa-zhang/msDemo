## 三、架构关键点
- **三层**：`DbManager.Common`（加密/日志/常量/错误翻译）、`DbManager.Core`（模型/接口/服务/**抽象层**/仓储）、`DbManager.Wpf`（MVVM 界面）。
- **跨库抽象层（Core/Abstractions + Core/Dialects）——所有跨库差异的唯一收敛点，禁止在 VM 里散写 `switch(dbType)`：**
  - `IIdentifierQuoter`：标识符引用（反引号/方括号/双引号）
  - `IDialect`：分页、限定名、时间/自增/拼接、**DDL（AddColumn/DropColumn/AlterColumn）**
  - `IDbTypeMapper` + `LogicalTypeEnum`：原生↔逻辑类型
  - `DialectProvider`：按 `DbTypeEnum` 取上述抽象
- **元数据**：`IDbMetadataService`（含 schema 维度）+ 5 库实现；`DbTreeNavigateService` 带 `MetadataCache`（懒加载+单节点刷新失效）。
- **本地存储**：SQLite（Dapper），`ConnRepository` 建表 + 字段迁移（迁移失败已容错，不阻断启动）。

## 四、已完成能力清单
### 连接
- 9 种类型枚举；6 种关系库连接串生成（**密码特殊字符已转义**）；未实现类型 UI 拦截（`DbTypeSupport`）
- 连接分组闭环：编辑窗**分配/新建分组**（可编辑下拉）→ 列表页**左侧分组树过滤** → 右键**删除分组（解组不删连接）**
- 连接测试、JSON 导入导出

### 导航树
- 层级：`连接→库→[Schema]→表/视图/存储过程/函数`（PG/SqlServer 有 Schema 层；Oracle 库即 schema；MySQL/SQLite 无）
- 表展开子组：**字段 / 索引 / 外键**
- 右键：查看数据、设计表、新建查询、截断表（走方言层）、复制名、**查看建表SQL（schema 感知）**、刷新
- 连接在线绿点、元数据缓存、搜索框（回车触发，仅顶层过滤）

### SQL 编辑器
- AvalonEdit 高亮、字体走设置
- 运行全部 / **运行选中（工具栏与 Ctrl+F5 均实时取选区）** / 停止
- **注释切换(Ctrl+/)、大小写切换(Ctrl+Shift+U/L)、轻量格式化器**
- **多结果集**多标签；双击结果单元格查看完整值
- **SELECT 自动限行**（设置可配，透明提示）
- 执行历史：**搜索 + 单条删除** + 清空（文件上限 500）

### 数据浏览编辑
- 分页、筛选（回车应用）、刷新
- 编辑：改单元格 / 新增行 / **删除选中行（真按选中）** / **撤销** / 保存（**UPDATE 按原始值定位、NULL 用 IS NULL、DML 走 ExecuteNonQuery**）
- 行号列、**列显隐**、**双击单元格查看完整内容（JSON 美化，共享 `CellValueViewer`）**
- 导出 Excel/CSV/INSERT/JSON/**HTML**；复制为 INSERT/CSV/Markdown/JSON
- 值格式化 `FormatValue`：数值用 InvariantCulture、BLOB→0x、Guid 加引号

### 表结构设计
- 加载字段、增删改字段属性、预览 + 执行
- **ALTER 生成下沉方言层**：各库正确语法（MySQL `MODIFY`、Oracle `MODIFY(...)`、SqlServer `ALTER COLUMN`+默认约束、PG/DB2 独立子句、SQLite 注释提示不支持）

### 全局
- 亮/暗主题、设置（字体/分页/限行/NULL 标记）、多标签、状态栏

- 导出连接 JSON 含明文密码（跨机导入需要，属设计取舍）
