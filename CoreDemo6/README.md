# ScheduleUpdater - 日程更新工具

这是一个简单的控制台应用程序，用于根据当前时间自动更新周六和周日的日程文件。

## 🎯 功能说明

该程序会根据当前时间距离基准日期（2026-01-04 23:59:59）的周数，自动判断是单周还是双周，并更新相应的日程文件。

### 周次判断规则
- **基准日期**: 2026-01-04 23:59:59（第一周）
- **单周**: 执行 "大理简单兵圣"
- **双周**: 执行 "大理困难天波府"

## 📁 文件路径

程序会自动更新以下两个文件：
- `C:\天龙小蜜\天龙小蜜\周六.txt`
- `C:\天龙小蜜\天龙小蜜\周天.txt`

**注意**: 文件字符编码为 **GB2312**

## 🚀 使用方法

### 1. 编译项目
```bash
cd ScheduleUpdater
dotnet build
```

### 2. 运行程序
```bash
dotnet run
```

或者直接运行编译后的可执行文件：
```bash
cd bin\Debug\net6.0
ScheduleUpdater.exe
```

## ⚙️ 工作原理

1. **计算时间差**: 计算当前时间与基准日期的天数差
2. **判断周次**: 根据 `weekNumber = (totalDays / 7) + 1` 计算周数
3. **确定任务**:
   - 单周（1, 3, 5...）→ "大理简单兵圣"
   - 双周（2, 4, 6...）→ "大理困难天波府"
4. **更新文件**: 将最后一行替换为新的日程信息
